문서뷰어팝업

미소지기 카드 및 마일리지 사용안내

매표

  • 전국 CGV 월 10회 무료 관람(본인 한정, 신분증 소지 필수)
  • 동일영화 2회 / 상영시간 10분전부터 / POS창 삭제 전까지 발권 가능
  • 잔여석 10%이상 남아있는 경우 발권 가능
    • ※ 사전좌석 확보 후 미소지기 카드로 재결제 불가
    • ※ 본인 외 관람 불가

매점

  • 매점 메뉴 할인 혜택
    (품목별 1일 2회/단, 팝콘, 탄산음료 사이즈별 1일 1회)
  • 음료 리필 불가(할인 구매)
  • 일부 사이트 기준 상이(위탁 등)
  • CGV 운영 투썸 40% 할인

기타

  • 포토티켓 월 10매 제작 가능
    (CJ ONE 통합회원 가입 필수/ 마일리지 발권 한정)
  • 마일리지 매표, 매점 사용 가능
    (3,000점 사용 시 영화발권 가능/ 본인 및 지인 관람 가능/근무, 휴식시간 매표 사용 불가)

특별관 사용 기준

  • 주중만 사용 가능: 4DX(월 2회), SPHERE X, SCREEN X
  • 주중/ 주말 사용 가능: 일반, IMAX(월 1회), STARIUM
  • 4DX/ SPHERE X/ SCREEN X 의 경우 마일리지 발권 불가
